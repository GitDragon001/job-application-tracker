# Job Application Tracker

A Python-based job application tracker that helps organize internship and job applications by company, role, deadline, and current status.

## Features
- Add new applications
- View all saved applications
- Update application status
- Delete applications
- Filter applications by status
- Save data using JSON

## Technologies Used
- Python
- JSON
- File handling

## How to Run
```bash
python3 main.py
